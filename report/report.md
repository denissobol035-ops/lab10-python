## 1. What was added

The following features were added:
- command-line interface using argparse
- file-based input/output
- JSON output format
- logging system

## 2. How the tool changed

The tool evolved from a demo-style script with hardcoded data
to a flexible CLI tool that accepts user input and works with files.

## 3. Why it matters

CLI improves usability and allows automation.

JSON provides machine-readable output for integration with other systems.

Logging improves debugging and transparency of execution.
